# Mobile sync guide

Use this project from a phone through GitHub.

Repository:
https://github.com/dlsgh87-spec/new-project

Best mobile workflow:
1. Open the GitHub app or mobile browser.
2. Go to `dlsgh87-spec/new-project`.
3. Open `PHONE-NOTES.md`.
4. Tap edit.
5. Write your note or task.
6. Commit changes to the `main` branch.

After you commit from the phone, both PCs will receive the change through automatic sync.

Important:
- Keep phone edits simple.
- Avoid editing the same file on two devices at the same time.
- If a conflict appears, stop and resolve it from a PC.
