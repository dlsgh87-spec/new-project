# 다른 컴퓨터와 같이 쓰는 방법

현재 이 폴더는 로컬 Git 저장소로 준비되어 있습니다.

## 현재 완료된 것

- Git 저장소 초기화 완료
- 기본 브랜치: `main`
- 첫 커밋: `Set up shared work threads project`
- 고정 업무 문서 4개 생성

## 다른 컴퓨터와 같이 쓰려면 필요한 것

두 컴퓨터가 같은 내용을 안정적으로 보려면 GitHub 같은 원격 저장소가 하나 필요합니다.

원격 저장소가 준비되면 이 프로젝트에서 아래 흐름으로 연결합니다.

```powershell
git remote add origin <원격 저장소 주소>
git push -u origin main
```

다른 컴퓨터에서는 같은 저장소를 받아서 Codex 프로젝트로 등록합니다.

```powershell
git clone <원격 저장소 주소>
```
